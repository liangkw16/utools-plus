# uTools Plus v1.1.0 发布材料

## 版本说明

v1.1.0 扩展 macOS 系统控制能力，新增声音与 Wi-Fi 管理，并替换原有扬声器占位入口。

- 新增「声音」命令：管理音频输出、音频输入、静音与音量调节。
- 声音音量步长调整为 5，拖动音量条时先本地更新，再节流写入系统，减少卡顿。
- 保留一键打开系统声音设置。
- 新增「Wi-Fi」命令：查看当前网络、已知网络、附近网络与扫描历史。
- 支持 Wi-Fi 开关、刷新、打开系统网络设置，以及切换到已知网络或输入密码连接新网络。
- 附近网络按名称合并展示，减少同名热点重复出现。

## 插件应用介绍

uTools Plus 是一个面向 macOS 的系统能力增强插件，用来把 uTools 中暂未直接覆盖、但日常高频使用的系统操作补上。

当前版本提供三个入口：

1. 蓝牙管理
通过「蓝牙」「bluetooth」「Bluetooth」唤起插件后，可以查看当前蓝牙状态、已配对设备列表，并对耳机、键盘、鼠标、触控板等设备执行快速连接或断开。常用设备可以收藏，后续会固定在列表顶部。

2. 声音管理
通过「声音」「sound」「音量」「静音」「输出设备」「输入设备」唤起插件后，可以查看音频输入/输出设备，切换设备，执行静音、取消静音和音量调节，也可以快速进入 macOS 声音设置。

3. Wi-Fi 管理
通过「Wi-Fi」「wifi」「无线网络」「网络」唤起插件后，可以查看当前 Wi-Fi 状态、当前网络详情、已知网络、附近网络和最近扫描历史。支持打开/关闭 Wi-Fi、刷新网络列表、打开系统网络设置，以及连接或切换网络。

适合场景：

- 经常在多个蓝牙设备、音频设备、无线网络之间切换。
- 希望用 uTools 命令直接处理系统设置，不想反复打开 macOS 设置面板。
- 想把蓝牙、声音、网络等系统能力集中到一个轻量工具入口。

使用说明：

- 本插件仅支持 macOS。
- 首次使用蓝牙能力时，请确保系统已完成设备配对。
- 使用 Wi-Fi 连接新网络时，受 macOS 权限和网络类型影响，部分网络可能仍需要在系统设置中完成认证。
- 原生系统能力依赖 macOS 系统接口，部分状态刷新可能受系统缓存影响，可点击「刷新」重新读取。
